title: Integrations
description: Connect GitHub to PipeWatch through a GitHub App installation and enable repositories for ingestion.

## GitHub App installation

PipeWatch stores one **integration** row per GitHub App installation. The integration links a workspace to an `external_installation_id` and tracks connection health.

## Repository enablement

After installation, admins choose which repositories PipeWatch monitors. Disabled repositories stop receiving new runs but retain historical data until removed.

## Tokens and refresh

PipeWatch exchanges installation credentials for short-lived GitHub API tokens server-side. Tokens are never exposed to the browser or stored in client storage.

## Re-install and suspend

If an installation is suspended or removed in GitHub, PipeWatch marks the integration inactive and surfaces a reconnect prompt in settings.
