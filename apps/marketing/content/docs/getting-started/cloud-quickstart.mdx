title: Cloud quickstart
description: Create a PipeWatch Cloud workspace, connect GitHub, and see your first workflow runs in minutes.
---

## Create your account

Sign up at **cloud.pipewatch.app** with GitHub OAuth. PipeWatch creates your first workspace automatically and assigns you the **owner** role.

## Connect GitHub

Install the PipeWatch GitHub App on the organizations or personal account you want to monitor. Grant repository access during installation — you can narrow scope later in GitHub settings.

## Add repositories

Open **Settings → Integrations** in PipeWatch and enable the repositories you want to track. New workflow runs appear on the dashboard within seconds via webhooks.

## Next steps

- Review [GitHub App Setup](/docs/github-app-setup/creating-the-app) if you need a custom app or self-hosted webhook endpoint.
- Browse the interactive [REST API reference](https://api.pipewatch.app/api/docs) for automation.
